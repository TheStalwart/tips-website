# Raspberry Pi 4 Model B

## Notable specs

- ARM64 64-bit architecture
- 1GB, 2GB, 4GB or 8GB LPDDR4-3200 SDRAM (depending on model)
- Gigabit Ethernet
- Wi-Fi 802.11ac
- Bluetooth 5.0
- 2x micro-HDMI outputs
- 15W (5V 3A) PSU required
